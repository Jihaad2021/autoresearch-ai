# AutoResearch AI

> Autonomous Research & Content Generation Platform using Meta Agent Architecture

## Overview

An intelligent multi-agent system that autonomously researches topics, generates comprehensive content, and ensures quality through collaborative review. 

### Architecture
- **7 Core Agents**: Controller, State Manager, Planner, Strategy, Orchestrator, Supervisor, Merger
- **17 Worker Agents**: Research, Analysis, Writing, Quality specialists
- **Framework**: LangGraph + LangChain
- **LLM**: Claude 3.5 Sonnet
